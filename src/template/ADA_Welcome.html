Te contamos más sobre este increíble proyecto

 

La aplicación móvil ADA consiste en un proyecto para generación automática de dictados musicales guiados por la configuración del profesor. La aplicación está especialmente diseñada para atender las necesidades de entrenamiento auditivo de estudiantes de las licenciaturas en interpretación musical. Se atiende a cubrir la necesidad de contar con una herramienta desarrollada en software libre que permita al profesor guiar el entrenamiento de sus estudiantes minimizando el esfuerzo dedicado a la producción de los dictados y maximizando las posibilidades de entrenamiento para los estudiantes. Se presenta la aplicación ADA desarrollada en JavaScript, de código abierto para la generación de dictados melódicos y rítmicos.
